# Expedition 0: Station Pulsar (backend)

Серверная часть для [Expedition-0](https://github.com/Vad1mChK/expedition-0), игры виртуальной реальности, обучающей
основам троичной логики.

Позволяет улучшить пользовательский опыт (UX) за счёт функций качества жизни (QoL), использующих сервисы искусственного
интеллекта. Однако игра полностью играбельна и без доступности сервера.
