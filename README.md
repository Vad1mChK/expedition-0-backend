# Expedition 0: Station Pulsar (backend)

## [RU]

Серверная часть для [Expedition-0](https://github.com/Vad1mChK/expedition-0-frontend), игры виртуальной реальности, обучающей
основам троичной логики.

Позволяет улучшить пользовательский опыт (UX) за счёт функций качества жизни (QoL), использующих сервисы искусственного
интеллекта. Однако игра полностью играбельна и без доступности сервера.

## [EN]

Server part for [Expedition-0](https://github.com/Vad1mChK/expedition-0-frontend), a VR game teaching the basics of ternary logic.

The server application aims to improve the user experience (UX) with quality-of-life (QoL) functions that use AI models.
However, the game remains fully playable even in autonomous mode.
